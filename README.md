# YouBike AI

這是一個結合 GIS與 AI的 side project，目標是利用臺北市 YouBike即時站點資料，建立站點風險地圖與簡單預測模型。

## 專案目標

- 抓取 YouBike 即時資料
- 顯示站點地圖
- 判斷缺車、缺位風險
- 建立簡單 AI模型做預測

## 技術工具

- Python
- Pandas
- GeoPandas
- Folium
- Streamlit
- Scikit-learn
- Git / GitHub
